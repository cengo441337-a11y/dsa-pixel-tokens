## LPC Standard-Format
- 4 Reihen: Unten (0), Links (1), Rechts (2), Oben (3)
- Typische Frame-Größe: 48×48px oder 64×64px
- Walk-Cycle: 3–9 Frames pro Richtung (meistens 3 oder 4)

## Test-Sprite laden
1. Sprite Sheet (PNG) in diesen Ordner legen, z.B. `hero.png`
2. Token in Foundry auswählen → Token konfigurieren → Tab "Pixel-Art"
3. Pfad eingeben: `modules/dsa-pixel-tokens/assets/hero.png`
4. Frame-Größe und Frames pro Richtung anpassen
